GOLANG on this SITE

The Golang is really easy to pick up. With an hour of practice I was able to push this site out using the Go App Engine SDK! I created a template using an existing prototype and gruntfile. The title of this site is stored in a type Data struct { Title string }. There is probably a better way to pass data to a template, but this was the fast road since it's taken straight out of the tutorial. Next I plan to write some callbacks and a singelton I can reuse throughout the app.

I love iOS!Obj-C==FOOD

I cloned Ray Wenderlichs tutorial about creating an interface similar to that of the AppStore. It was a bit outdated so I removed his calls to [obj release] and some other elements that are no longer needed with ARC. I then updated some of the styling code and adpated it to work with a Storyboard instead of Nibs. My final result does practically the same thing as Rays, but instead of just scrolling right and left, my cells snap into place. Examine the smooth affect the snapping has, which can be adjusted and note how the cells act when the tableView has reached the last cell. The code does not snap when the last cell is "partially" showing, but when the last cell is "mostly" showing, it snaps.
